# PrimeFactors_Functions
Three functions in Scheme to break a number down to its prime factors, generate a list of count-element pairs given a list, and to take in a list of count-element pairs to give a number. The three functions in succession will result in no change.

Note that this was an academic assignment. That means that the function headers as well as what program should do were given by the professor and to be carried out by students, or me.
